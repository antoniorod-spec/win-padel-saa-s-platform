Importación por Categoría

El administrador selecciona una categoría específica en la plataforma antes de subir el archivo. La columna 'Categoría' en el Excel es ignorada.

info Columna 'Categoría' es opcional
language

Importación Masiva Global

Carga todos los niveles en un solo archivo. El sistema mapea automáticamente a cada pareja según la columna 'Categoría'.

check_circle Mapeo automático por nivel
lightbulb

Lógica de Identidad: El sistema utiliza el Número de Teléfono para identificar usuarios existentes y asignarles su RPM-ID histórico. Si el teléfono no existe, se creará un nuevo perfil de jugador.

rule Reglas de Validación

  • check_circle

    Campos Obligatorios

    Nombre y Teléfono de ambos jugadores son mandatorios para procesar la pareja.

  • phone_iphone

    Identificador Único (Teléfono)

    El número telefónico es la llave primaria (Primary Key) para identificar usuarios y asignar RPM-IDs. No puede repetirse para distintos nombres.

  • check_circle

    Formatos Admitidos

    Archivos .xlsx o .csv. Se recomienda usar el código de país (ej. +52).

  • warning

    Categorías Exactas

    Para la importación global, el texto de la categoría debe coincidir con el catálogo del torneo.
